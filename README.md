## üöÄ Welcome to Nispo Track

**Nispo Track** is a powerful, open-source 3D space visualization tool designed to turn your browser into a mission control center. Whether you are an astronomy enthusiast, a student, or an aerospace professional, Nispo Track puts thousands of satellites, debris, and celestial objects at your fingertips.

Experience the orbital mechanics of our planet in real-time. Witness the dance of megaconstellations, analyze potential collisions, and explore the history of space launches with stunning visual fidelity.

## ‚ú® Key Features

*   **Real-time Satellite Tracking**: Track over 30,000 objects in real-time with high-precision SGP4 propagation.
*   **Immersive 3D Visualization**: Navigate a beautiful 3D Earth with realistic atmosphere, night lights, and celestial backgrounds.
*   **Collision Analysis**: Visualize potential conjunctions and breakups.
*   **Launch History**: Travel back in time to glimpse historical satellite launches.
*   **Pro-Level Tools**: Access advanced sensor views, field-of-view projections, and look-angle calculations.
*   **Mobile Friendly**: Designed to work seamlessly on desktops, tablets, and mobile devices.

## üõ†Ô∏è Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm (v9 or higher)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Pownkumar1234/powney---3d-satelite-toolkit.git
    cd nispo-track
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # Note: If you encounter issues with submodules, ensure you initialize them:
    git submodule update --init --recursive
    ```

3.  **Start the development server**
    ```bash
    npm run start
    ```

    OPEN [http://localhost:8080](http://localhost:8080) to view it in your browser.
